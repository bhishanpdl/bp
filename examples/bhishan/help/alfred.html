Help for Alfred

- cm 2 ==> clip merge last 2 clipboards
- nfc a.py ==> create a.py in pwd with content from clipboard
- gh Alfred ==> open Alfred GitHub repo
- tut pandas ==> open tutorial pandas GitHub repo.
- qq ==> open clipboard in Fenetre.
- dcalc now + 2M ==> now plus 2 minutes (set your date format mm.dd.yy)
